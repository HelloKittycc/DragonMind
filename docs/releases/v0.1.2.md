## Summary

v0.1.2 adds Recurring Review Context Foundation to DragonMind, anchored on monthly business review.

It introduces a structured review workflow where DragonMind acts as a review advisor:

- Prepares the monthly review context
- Aggregates only user-confirmed inputs
- Proposes traceable questions worth analyzing
- Converts questions into Nodes only after explicit user confirmation

## Boundaries Preserved

The core model boundaries remain:

- Review Session is not Node
- Topic is not project management
- Review Section is not a conclusion
- Guiding Question does not answer itself
- Guiding Question does not create Node automatically
- Evidence remains attached only to Node or Relation
- Knowledge inclusion is not Evidence

v0.1.2 intentionally does not add:

- Automatic report generation
- Automatic monthly report generation
- Automatic next-month plan generation
- Automatic Evidence creation
- Automatic Task creation
- RAG
- Vector DB
- Embeddings
- Semantic search
- Full-library scans
- Automatic fact judgment

## Monthly Lifecycle

Active monthly Topics can automatically prepare the current month's Review Session. Creation is deterministic and idempotent.

Automatic preparation creates only an empty Review Session and its seven default Review Sections. It does not create Nodes, Tasks, Evidence, Guiding Questions, inputs, or conclusions.

Duplicate Review Sessions for the same Topic and business period are prevented at both the service layer and the database unique-index layer.
