## Summary

v0.1.1 adds Knowledge Ingestion / Executive Document Ingestion to DragonMind while preserving the graph-first model.

This release extends v0.1 by allowing external materials to be imported, searched, and explicitly cited as Evidence. Knowledge remains background material. It does not become a Node, does not enter the Relation graph, and does not automatically create conclusions.

## What's Included

- `knowledge_source` / `knowledge_chunk` data layer
- `evidence.knowledge_chunk_id`
- Paste import
- File upload
- Parser adapters for PDF, DOCX, PPTX, XLSX, CSV, TXT, MD, JSON
- Text-based PDF only; no OCR
- Deterministic extraction and chunking
- SQLite LIKE search
- `/knowledge` page
- AdvisorDrawer
- Knowledge search and citation UI
- Explicit chunk-to-Evidence flow
- Evidence panel showing "来自资料库"
- Final Knowledge UX design archive
- Final Knowledge Ingestion spec update

## Boundaries Preserved

v0.1.1 intentionally does not add:

- RAG
- Vector DB
- Embeddings
- Semantic search
- OCR
- Automatic fact judgment
- Automatic Evidence creation

The core model boundaries remain:

- Knowledge is not Node
- Knowledge does not enter Relation graph
- Evidence remains explicit user citation
